# DeusCaritasEst
"God is Love" -- A Daily Devotion Tracking App for Catholics \
I hope that this service is used as a tool to foster each faithful's daily prayer life, rather than simply checking off boxes. \
I made it because I wanted to keep track of how much I attend Mass, pray the Rosary, and spend time with Jesus in other ways during school breaks when I'm not spending my 24/7 at the Newman Center :> \
